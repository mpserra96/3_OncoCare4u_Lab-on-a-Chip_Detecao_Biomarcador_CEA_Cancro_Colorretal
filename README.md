# 3_OncoCare4u_Lab-on-a-Chip_Detecao_Biomarcador_CEA_Cancro_Colorretal
Neste repositório, encontra-se o estudo conceptual do OncoCare4u, um dispositivo lab-on-a-chip para deteção do biomarcador CEA no cancro colorretal, realizado por mim e três colegas no âmbito da UC de Projeto Integrador da Licenciatura em Engenharia Biomédica, da Universidade do Minho.
